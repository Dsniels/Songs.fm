# Songs.fm

Android app to discover new music, interesting facts about artists and songs Using the Spotify Web API, Genius API, inspired by tinder cards

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
